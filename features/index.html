Features

  1. Product Management

    Easily manage and organize your product catalog. Add new products, update existing ones, and manage stock levels to ensure you're always prepared for customer demands. The system supports detailed product descriptions, pricing, and category assignments.

  2. Discount Code Management

    Create and manage discount codes to offer promotions to your customers. Set up custom discount rules (e.g., percentage off, fixed amount off) for specific products or categories, and track which codes have been used. This feature helps boost customer engagement and increase sales.

  3. Inventory Management

    Keep track of your stock levels and make adjustments as needed. The system notifies you when stocks are low, ensuring that you never run out of popular items. You can also update inventory based on stock movements (e.g., sales, returns, or restocking).

  4. Role-Based Access

    The system supports multiple user roles, allowing for different levels of access to various features. Store admins can manage product data, sales, and inventory, while sales associates may only handle customer transactions and basic inventory management.

  5. Order History

    Customers can view their order history, including order details and shipping status. This feature enhances customer satisfaction by providing easy access to previous purchases, helping them track past orders or repeat purchases.

Tech Stack

  • Backend: Go (Golang)
  • Web Framework: Fiber
  • Database: MySQL

API Endpoints Documentation

List of available endpoints for the URL Shortener Application.

Barang Routes

Method Endpoint Description
GET /barang Fetch all barang (products)
GET /barang/:id Fetch a specific barang by ID
POST /barang Create a new barang
PUT /barang/:id Update a barang
PUT /barang/stok/:id Update stock of a barang
DELETE /barang/:id Delete a barang

Penjualan Routes

Method Endpoint Description
GET /admin/penjualan Fetch all penjualan (sales) records
GET /admin/penjualan/:id Fetch a specific penjualan by ID
POST /penjualan Insert new penjualan data

Kode Diskon Routes

Method Endpoint Description
GET /admin/kode-diskon Fetch all kode diskon (discount codes)
GET /admin/kode-diskon/:id Fetch a specific kode diskon by ID
GET /kode-diskon/get-by-code Fetch a kode diskon by its code
POST /kode-diskon Create a new kode diskon
PUT /admin/kode-diskon/:id Update a kode diskon
DELETE /admin/kode-diskon/:id Delete a kode diskon

Installation and Setup

  1. Clone the repository: git clone https://github.com/Pande17/toko-retail.git
  2. Navigate to the directory: cd shortlink
  3. Install dependencies: go mod tidy
  4. Create a .env file with the necessary environment variables.
  5. Run the application: go run main.go
  6. Access the application at http://localhost:3000
